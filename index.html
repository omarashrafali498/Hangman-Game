Welcome to Hangman Game

Hangman is a classic word-guessing game where players try to guess a hidden word by suggesting letters within a certain number of attempts. The game is typically played with a drawing of a gallows and a stick figure representing the hangman. Each incorrect guess results in adding a part to the hangman figure, and the game ends when the figure is fully drawn or the word is guessed correctly.


Objective

The objective of Hangman is to guess the hidden word before running out of attempts. The game tests your vocabulary and guessing skills while providing an entertaining challenge.
